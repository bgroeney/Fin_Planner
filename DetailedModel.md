
Key functionality should be:
+ Table should be spreadsheet like, with each line item on a new row and columns being teh relevant period.
+ Cash flow calculations and line items including Gross Rent, Vacancy allowance, Outgoings (Council Rates, Water Rates, Insurance, Repairs & Maintenance, Management Fee, Other), Debt Servicing
+ Key calculated fields - Net Rental Income, Net Operating Expenses, Net Operating Income, Net Cashflow, Net Depreciation, Net Taxable Income
+ Ability to add additional Revenue and Expense line items
+ Ability to switch between Yearly and Monthly View. Yearly should sum Monthly when there is a monthly override. Yearly values should be locked when calculated from Monthly values. Show as Read only with a padlock.
+ Navigation of table should be possible via keyboard and prevent left scroll navigating page backwards
+ Right clicking on a cell should allow copy left and copy right options
+ Right clicking Edit should allow cell value to be set with a distribution of uncertainity and growth rate. Further, Edit should support a formula based input for all cell inputs i.e. [Gross Rent] * 0.08 to calculate something like Management Fee.
+ Clear visual representation where values are manually set vs pre-populated or calculated
+ Values should be locked when manually overwritten from any further updates from the Assumptions and Inputs tab. Further, teh cell should need to be unlocked to allow updates, either by individual selecting and adjusting cell, using global Unlock All, or using Copy Left or Copy Right
